## Project Summary
An AI-powered app that:
  - Recommends music based on the user's mood or what they write about how they feel.
  - Plays 30-second preview tracks to showcase different genres.
  - Provides song recommendations with links to Spotify,YouTube, and Apple Music..etc.

  ## Main Features
  1- Mood or Text-Based Music Input.
  2- Genre Exploration.
  3- Audio Preview For Each Genre.
  4- Each Recommended Song Includes Direct Links to:
  . Spotify
  . YouTube
  . Apple Music
  5- Save favorite songs and create playlists.
  6- Song Upload & Similar Track Founder.
  7- Daily music recommendations.

## AI Functionality (Gemini)
- Takes the user's mood as input, either by selecting a mode or typing it in.
- Accepts the selected or written genre and shows a 30-second preview track for each genre.
- Suggests songs based on mood and genre, providing URLs for Spotify, Youtube, and Apple Music.
- Recommends daily music picks tailored to the user's mood and preferences.

## API
- Gemini API.

## Users Flow
- Users select a mode from a list or enter it manually.
- Users select a music genre.
- Users can display previews to discover genres.
- Users can click on a song link to listen on Spotify, YouTube, or Apple Music.
- Users can upload a song to find similar tracks.
- Users can add songs to favorites or create custom playlists.
