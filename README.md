# CSHub-Hack-Day
This is our project for CSHub Hack Day

Mood-Based Music Recommendation APP
An AI-powered music recommendation app that analyse the user's mood or text input to generate personalized music suggestions. The app supports genre exploration,
preview tracks, playlists, and links to major streaming platforms.

Project Summary

This application uses AI to understand the user's emotional input and recommend songs that fit their mood. Users can browse music genres, listen to 30-second
previews, save favorite tracks, create playlists, andd upload songs to find similar tracks.

Features

Mood or Text-Based Input
. Users can select a mood or describe how they feel.
. AI suggests songs based on the emotional context.

Genre Exploration
. Browse multiple music genres.
. Each genre includes a 30-second audio preview.

Song Recommendation Links
Each recommended song includes links to:
. Spotify
. YouTube
. Apple Music

Favorites & Playlists
. Save songs to a favorites list.
. Create and manage custom playlists.

Song Upload & Similar Track Finder
. Upload a song file.
. AI analyzes the audio and suggests similar tracks.

Daily Recommendations
. Daily personalized music suggestions based on user preferences.


AI Functionality (Gemini API)
The app uses the Gemini API to:
. Process mood or typed emotional descriptions
. Provide genre suggestions and preview tracks
. Recommend songs with streaming platform URLs
. Generate personalized daily music picks

API Used
. Gemini API

User Flow
1. User selects or types a mood.
2. User chooses a genre.
3. App plays a 30-second preview of each genre.
4. AI generates music recommendations.
5. User opens recommended tracks on Spotify, Youtube, or Apple Music.
6. User may upload a song to find similar tracks.
7. User saves songs or builds playlists.
